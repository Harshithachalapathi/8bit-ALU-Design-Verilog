# 8bit-ALU-Design-Verilog
De# 8-bit ALU Design using Verilog HDL

## Project Description

This project implements an **8-bit Arithmetic Logic Unit (ALU)** using Verilog HDL.
The ALU performs arithmetic and logical operations on two 8-bit inputs.

## Operations Implemented

* Addition
* Subtraction
* AND
* OR
* XOR
* Shift Left
* Shift Right

## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
* EDA Playground

## Skills Demonstrated

* Digital Logic Design
* Verilog HDL
* Hardware Simulation
* Digital Circuit Verification
sign and simulation of 8-bit ALU using Verilog HDL
